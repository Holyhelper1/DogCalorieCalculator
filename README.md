# 🐶 Калькулятор корма для собаки 🥣

Этот проект представляет собой приложение, которое помогает рассчитать количество корма, необходимого для вашей собаки, в зависимости от её возраста, веса и активности. 🐕✨

## 💡 Описание

Калькулятор позволяет вам ввести:
- 📅 **Возраст** вашей собаки (0-3 месяца, 3-6 месяца, 6-12 месяцев, 1-2 года и 2 года и старше).
- ⚖️ **Вес** вашей собаки в килограммах.
- 🏃 **Активность** вашей собаки (количество физических активностей в день).
- 🍽️ **Калорийность** корма на килограмм (базовое значение — 4270 ккал/кг).

На основе этих данных приложение рассчитывает количество корма в граммах, которое нужно давать вашей собаке в день.



## Новое описание:
📄 Документация проекта: "DogFood Calc"
1. Цель проекта
Создание мобильного приложения для расчета суточной нормы кормления собак на основе веса, возраста, активности и калорийности корма.
2. Функциональные требования
Ввод данных: Выбор возраста (пресеты), ввод веса (кг), ввод калорийности (ккал/кг), выбор уровня активности.
Расчет: Автоматический пересчет при изменении любого поля (или по кнопке).
Локализация: Поддержка RU/EN языков (использование библиотеки i18next).
Адаптивность: Mobile-first дизайн, оптимизированный под сенсорные экраны.
3. Технологический стек
Framework: React 18+ (TypeScript).
Build Tool: Vite.
Styles: CSS Modules (чистый CSS или SCSS).
State Management: React Context API (для хранения настроек и языка).
Mobile Bridge: Capacitor (для доступа к нативным функциям и публикации в Store).

## 🛠 Требо
🎨 Рекомендации по UX/UI (Modern Mobile)
Чтобы приложение выглядело "дорого" на смартфонах, примените эти CSS-приемы:
Инпуты: Вместо стандартных границ используйте легкую заливку фона и фокус-состояние с мягким свечением.
Типографика: Системные шрифты (-apple-system, BlinkMacSystemFont) выглядят нативно.
Сенсорные зоны: Все кнопки должны иметь высоту не менее 48px для удобного нажатия пальцем.
Скругления: Используйте border-radius: 12px или 16px для карточек и кнопок.
📲 Путь в App Store / Google Play (через Capacitor)
Так как вы пишете на React, самый быстрый путь в сторы — Capacitor.js от команды Ionic. Он превращает ваш веб-билд в нативное приложение.
Шаги интеграции:
Установка: npm install @capacitor/core @capacitor/cli.
Инициализация: npx cap init.
Сборка проекта: npm run build.
Добавление платформ:
npx cap add android
npx cap add ios
Запуск: Он создаст нативные проекты для Android Studio и Xcode, где вы сможете настроить иконки, сплэш-скрины и собрать финальный файл для публикации.
